# koudaiii.com-terraform


## Setup

1. [Authenticating using a Service Principal and OpenID Connect](https://registry.terraform.io/providers/hashicorp/azuread/latest/docs/guides/service_principal_oidc)
2. [Install tfenv](https://github.com/tfutils/tfenv)
3. [Configuring a User or Service Principal for managing Azure Active Directory](https://registry.terraform.io/providers/hashicorp/azuread/latest/docs/guides/service_principal_configuration)
4. [Store Terraform state in Azure Storage](https://learn.microsoft.com/ja-jp/azure/developer/terraform/store-state-in-azure-storage?tabs=azure-cli)
5. [Install tflint](https://github.com/terraform-linters/tflint)
6. [Install TFLint Ruleset for terraform-provider-azurerm](https://github.com/terraform-linters/tflint-ruleset-azurerm)
7. [Install tfcmt](https://suzuki-shunsuke.github.io/tfcmt/)
